Getting Started

Installation

Install ngx-analog-clock using npm:

npm install @craftedcode-dev/ngx-analog-clock

Quick Start

Import the component in your standalone component or module:

import {{ '{' }} NgxAnalogClockComponent {{ '}' }} from 'ngx-analog-clock';

@Component({{'{' }}
  selector: 'app-root',
  standalone: true,
  imports: [NgxAnalogClockComponent],
  template: '<ngx-analog-clock></ngx-analog-clock>'
{{ '}' }})
export class AppComponent {{ '{' }}{{ '}' }}

That's it! You now have a working analog clock.

Timezone Support

Display time from any timezone using IANA timezone strings. Perfect for world clocks and international applications.

clockConfig: AnalogClockConfig = {{'{' }}
  size: 250,
  timezone: 'America/New_York',
  label: {{'{' }}
    text: 'New York',
    position: 'bottom'
  {{ '}' }}
{{ '}' }};
Supported Timezones: All IANA timezone strings are supported (e.g., "Europe/Paris", "Asia/Tokyo", "Australia/Sydney"). Use "local" for the user's system timezone.

Custom Time Control

Control exactly what time the clock displays using the time property in config. Perfect for timers, countdowns, or displaying specific moments.

export class TimerComponent implements OnInit {{'{' }}
  customTime: Date = new Date();

  get clockConfig(): AnalogClockConfig {{'{' }}
    return {{'{' }}
      time: this.customTime,
      size: 300,
      theme: 'light'
    {{ '}' }};
  {{ '}' }}

  ngOnInit() {{'{' }}
    // Update every second
    setInterval(() => {{'{' }}
      this.customTime = new Date(this.customTime.getTime() + 1000);
    {{ '}' }}, 1000);
  {{ '}' }}
{{ '}' }}

<ngx-analog-clock [config]="clockConfig"></ngx-analog-clock>
Note: When time property is set in config, the clock displays that time instead of the current time. Update the Date object to animate the clock.

Timer / Countdown

Create a countdown timer or stopwatch.

  targetTime: Date;
  currentTime: Date = new Date();

  ngOnInit() {{'{' }}
    // Set countdown target (e.g., 10 minutes from now)
    this.targetTime = new Date(Date.now() + 10 * 60 * 1000);
    
    setInterval(() => {{'{' }}
      const remaining = this.targetTime.getTime() - Date.now();

Performance

Optimized Rendering

The clock uses efficient rendering techniques:

  • 60 FPS animations - Smooth second hand movement with requestAnimationFrame
  • OnPush change detection - Minimal re-renders for better performance
  • Pure CSS animations - Hardware-accelerated transforms
  • SVG-based - Crisp rendering at any size without pixelation

Best Practices

  • Use smooth: false if you have many clocks on one page to reduce CPU usage
  • Avoid updating [time] input more than once per second
  • Consider hiding seconds hand (hands.second.show: false) for static displays
  • Use appropriate size values - larger clocks use more resources
